# Tutorial filtrado por fechas — ASP.NET Core MVC

En este video se explica cómo crear un proyecto con ASP.NET Core MVC y cómo filtrar envíos por rango de fechas para calcular el costo total por repartidor. El sistema usa tres tablas principales: Repartidor, Envios y Zonas. La lógica multiplica el peso de cada envío por la tarifa por kilogramo de su zona y luego muestra el resultado agrupado por repartidor y zona.
